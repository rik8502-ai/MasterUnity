# MasterUnity

Unity 6.4 で制作した 2D アクションプラットフォーマーのポートフォリオ作品です。

## 動作環境

| 項目 | 内容 |
|------|------|
| Unity バージョン | **Unity 6.4 (6000.4.5f1)** |
| レンダーパイプライン | 2D |

> 上記バージョン以外では正常に動作しない場合があります。Unity Hub から **6000.4.5f1** をインストールしてプロジェクトを開いてください。

## プレイ方法

1. リポジトリをクローンし、Unity Hub から本プロジェクトを開く
2. **File > Build Settings** を開き、先頭シーンが `Assets/Scenes/TitleScene.unity` になっていることを確認する
3. エディタ上部の **Play ボタン** を押してプレイ開始

**タイトルシーン（TitleScene）からスタートしてください。**  
Build Settings の先頭シーンとして登録されているため、通常は Play ボタンを押すだけでタイトル画面から開始されます。別のシーンを開いた状態で Play した場合は、`Assets/Scenes/TitleScene.unity` を開いてから再生してください。

## 操作

| 操作 | キー / 入力 |
|------|-------------|
| 移動 | A / D キー、または左右矢印キー |
| ジャンプ | W キー または Space キー |
| 攻撃 | 左クリック（近接：クラブ / 遠距離：ファイアーボール） |

## シーン構成

| 順番 | シーン | 内容 |
|------|--------|------|
| 1 | TitleScene | タイトル画面 |
| 2 | MainScene | ステージ 1 |
| 3 | Stage2 | ステージ 2 |
| 4 | BossStage | ボスステージ |

## 主な実装内容

- プレイヤーの移動・ジャンプ・攻撃（近接 / 遠距離）
- 敵 AI（移動・攻撃・HP 管理）
- シーン間の HP 引き継ぎ（GameManager）
- UI（HP 表示、ゲームオーバー、ステージクリア）
- サウンド管理
